# Flappy Clone

Ein kleiner Flappy-Bird-Klon als statische Canvas-App.

## Starten

```bash
cd /Users/jona/Desktop/githubi/githubii
python3 -m http.server 4173
```

Dann `http://localhost:4173` im Browser oeffnen.

## Steuerung

- `Leertaste` oder `Klick/Tap`: flattern / neu starten
- `f`: Vollbild umschalten
- `Esc`: Vollbild verlassen
