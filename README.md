# Flappy POV

Ein Flappy-Bird-inspirierter Browser-Runner in hochwertiger 3D-POV-Inszenierung mit `Three.js` und `Vite`.

## Starten

```bash
cd /Users/jona/Desktop/githubi/githubii
npm install
npm run dev -- --host 127.0.0.1 --port 4173
```

Dann `http://localhost:4173` im Browser oeffnen.

## Build

```bash
npm run build
```

## Steuerung

- `Leertaste`, `Pfeil hoch` oder `Klick/Tap`: Schub nach oben / neu starten
- `f`: Vollbild umschalten
- `Esc`: Vollbild verlassen
